# Zombies Dots

A Minecraft 1.8.9 Forge client mod for placing precise dot markers on block faces to help mark wallshots in Hypixel Zombies. Dots are stored per profile so you can keep separate sets for different maps.

## Commands

| Command | Description |
|---|---|
| `/adddot` | Aim at a block and place a dot marker |
| `/removedot` | Remove the dot closest to your crosshair on the aimed block |
| `/dotgui` | Open the profile manager |
| `/profile <name\|off>` | Switch active profile, or use `off` to hide all dots |
| `/profiles` | List all profiles with marker counts |
| `/dotcolor <RRGGBB>` | Set the custom color slot (hex, e.g. `FF8800`) |
| `/dothelp` | Show all commands in game |

## Installation

Place the `.jar` into your `.minecraft/mods` folder. Requires [Minecraft Forge](https://files.minecraftforge.net/) for 1.8.9.
